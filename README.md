
# 魔宫迷途

你，99 级的勇者，已经站在了魔宫的入口。

魔王，万物之恶的集合体，如今已近在眼前。

你早已为今天做好了准备。

这些怪物对早已身经百战的你而言，根本不值一提。

但事情并不会一帆风顺……

---

* 音乐：来自《Monster Hunter: World》

本游戏制作于 GLOBAL GAME JAM 2021，一个两天内从头到尾开发一个游戏的活动，这一届的主题是 LOST AND FOUND。
